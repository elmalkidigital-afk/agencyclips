# **App Name**: HadithTok Studio

## Core Features:

- AI Hadith Generation: Generate authentic Hadith content from Bukhari & Muslim sources using AI tool.
- Customizable Background: Allow users to upload custom background images for their Hadith videos or select a curated random image.
- Text Overlay Animation: Animate the Hadith text and source to create an engaging visual experience. Dynamically adjust speed based on number of words.
- Video Generation: Generate a .webm video of the animated Hadith for TikTok.
- Video Download: Allow users to download the generated video file.

## Style Guidelines:

- Primary color: Rose-red (#FE2C55) evokes a sense of importance and authenticity, tying into the cultural reference of the app.
- Background color: Very dark desaturated red (#161616) creates a modern backdrop.
- Accent color: Cyan (#25F4EE) contrasts against the rose and improves readability.
- Body and headline font: 'Segoe UI' (sans-serif) for a modern and readable look on mobile devices. Note: currently only Google Fonts are supported.
- Use simple, clean icons for actions like generate, download, and post to TikTok. Consistent with the circular icons that are shown in the sample document.
- Maintain a clean and straightforward layout, with controls on the left and video preview on the right.
- Implement smooth scrolling animations for the Hadith text, ensuring readability and engagement.
